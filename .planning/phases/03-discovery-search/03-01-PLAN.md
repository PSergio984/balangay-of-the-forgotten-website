# Phase 3 Plan: Discovery & Search

## Goal
Implement a professional Search Hub and an interactive visual timeline of the game's lore.

## Success Criteria
1. Search Hub (`/wiki/search`) implemented with collection filtering.
2. Timeline (`/wiki/timeline`) implemented with chronological lore events.
3. Image loading delay eliminated (under 500ms).
4. Global search results include snippets and category tags.
